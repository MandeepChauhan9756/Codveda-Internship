# Number Guessing Game

## Description
This is a simple number guessing game made in Python. 
The program generates a random number between 1 and 100, and the user has to guess it.

## Features
- Random number generation
- User input from terminal
- Gives hint like "Too High" or "Too Low"
- Keeps running until correct guess

## How to Run
Run the following command in terminal:

python game.py
