# Word Counter

## Description
This is a basic Python program that reads a text file.  
The user enters the file name, and the program tries to open and read it.

## Features
- Takes file name as input
- Reads the file content
- Shows message if file is not found
- Handles simple errors

## How to Run

1. Save the code as `word_counter.py`
2. Open terminal in the same folder
3. Run:

python word_counter.py

4. Enter the file name (for example: sample.txt)

## Note
- Keep the text file in the same folder
- If the file name is wrong, it will show an error
